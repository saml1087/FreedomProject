Hardware Components


There are several components within a computer. Each component is dedicated to perform a certain task. Some of these components include: (Click on Names For More Info)

The circuit board, presented in many sizes (such as ATX, mini-ITX, micro-ATX, EATX) allow the communication between components that's plugged into it. These include the processor, any case front panel connectors, RAM, Graphics Card, and many more.

The processor, or the CPU, plays a crucial part in a computer system. Typically, some processors contain an integrated graphics card (which means that it has a GPU built into it, performing graphic related calculations), but there are also processors do not have an integrated graphics card. The processor itself is basically the brain of the system. It allows for calculations to be performed and its efficiency can be defined by its cores, threads, and clock speed.

A CPU Cooler is exactly what it's called. In order to keep a CPU cool in terms of temperature, it cannot become overwhelmed which leads to performance lost. It can be in a AIO Liquid Cooler, or a regular Fan Cooler. Note that some processors come with a stock cooler which typically is a Fan Cooler.

Random Access Memory is crucial as it is the component that controls the amount of tabs you have on. Typically, if you're running several tabs all the time, it's better to have higher storage of RAM. This basically allows the CPU to take information quickly from the RAM instead of the Hard Drive (which would be a lot slower). It can also have a speed attached to it. Typically regular computers have around 2133 MHz of RAM, but higher end PC's have around 3000 MHz at least.

The Graphics Card (also known as Video Cards or GPU) is the card that handles heavy graphics running in your monitor. As some CPU's come with integrated GPUs, this is only necessary if you would like to run Higher Quality Games with High Graphics. Even if your processor comes with an integrated graphics card, it's still recommended to get a separate GPU if you're trying to run high graphics games.

The Power Supply basically powers your whole computer. When placed in the case, it has multiple wires that connect to the motherboard and the components in your build. The PSU will then be plugged into an outlet which allows power to run through the PC.

The HDD and SSD are storage devices that holds all your files and games. Typically for faster boot times, it's recommended to get an SSD since it's made out of electronical parts unlike a Hard Drive that uses mechanical parts to access data. Hard Drives tend to be more storage for lower speeds, while SSD's are higher speed for lower storage.

The Case holds all of your components together. Typically, your case should have fans that allow airflow to be positive, meaning all of the heat will leave the case, with more air generated inside the case.Some Cases come with a thermal glass side panel which allows you to see inside. It's also a nice touch to decorate a case but that's really optional. Overall, it's not necessary, but the fans should have both exhaust and intake fans.



Note that these excludes peripherals (keyboards, mouse, etc)
